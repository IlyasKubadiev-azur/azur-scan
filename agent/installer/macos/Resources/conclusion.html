Installation complete

The agent is now installed, configured against the office backend, and running as a system daemon.

This Mac will appear in the Azur-Scan admin panel within ~60 seconds. No further action required.

Where to look if something is wrong

  • Logs: /Library/Logs/AzurScan/agent.log
  • Config: /Library/Application Support/AzurScan/config.yaml
  • Service status: sudo launchctl print system/com.azurscan.agent

To point this Mac at a different backend later: sudo /usr/local/azurscan/bin/azurscan-agent set-config --server <URL> then sudo launchctl kickstart -k system/com.azurscan.agent.
