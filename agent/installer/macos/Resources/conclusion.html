Installation complete

The agent is now installed and running as a system daemon.

Point it at the backend

Open Terminal and run:

sudo /usr/local/azurscan/bin/azurscan-agent set-config \
     --server http://10.0.20.143:8000
sudo launchctl kickstart -k system/com.azurscan.agent

The daemon will auto-register this Mac on the next start — no enrollment token needed; the backend identifies the device by its stable IOPlatformUUID.

Logs: /Library/Logs/AzurScan/agent.log
